Run your application with zero overhead

IncludeOS allows you to run your application in the cloud without an operating system. IncludeOS ads operating system functionality to your application allowing you to create performant, secure and resource efficient virtual machines.

IncludeOS applications boot in tens of milliseconds and require only a few megabytes of disk and memory.

Get started

Installing IncludeOS on Linux or OS X typically takes around 10 minuttes. Once installed we suggest looking at and booting a few of the examples to familarize yourself with the system. 

export INCLUDEOS_SRC=~/IncludeOS
export INCLUDEOS_PREFIX=~/IncludeOS_install
cd $HOME
git clone https://github.com/hioa-cs/IncludeOS.git
cd $INCLUDEOS_SRC
./install.sh

Once installed have a look in ~/IncludeOS/examples/demo_service. In order to boot the application just type "boot --create-bridge ." and the applcation should build and run.

We strive to make it easy to create fast and useful services. The below code will set up a simple "Hello World" TCP service and happliy talk to anyone connecting.

#include <os>
#include <iostream>
#include <net/inet4>

void Service::start(const std::string& args) {
    auto&& inet = net::Inet4::ifconfig<0>(30.0);  // Initialize IP stack
    auto&& server = inet.tcp().bind(80); // Set up a TCP server on port 80
    for (auto&& ref : collection) {
        std::cout << ref.to_string() << "\n";
    }
    server.on_connect([](auto conn) {
        conn->on_read(1024, [conn](net::tcp::buffer_t buf, size n) {
            conn->write("Hello world\n", [](auto size) {
                std::cout << "Response written\n";
            });
        });
    });
}
