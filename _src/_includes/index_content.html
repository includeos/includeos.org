Run your application with zero overhead

IncludeOS allows you to run your application in the cloud without an operating system. IncludeOS ads operating system functionality to your application allowing you to create performant, secure and resource efficient virtual machines.

IncludeOS applications boot in tens of milliseconds and require only a few megabytes of disk and memory.

Performant. No operating system overhead.

Perfectly portable. Runs on all major hypervisors.

Secure. No unused code in your image.

Lean. Base virtual image starts at about 1 megabyte.

IncludeOS is a Unikernel

Unikernels are a new class of operating systems. The functionality in the operating system is available as a library. The build systems adds the operating system into the application itself. You can read more about unikernels in general and other unikernel implementations at unikernel.org.
